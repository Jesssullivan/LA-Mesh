# Node Installation Toolkit

Bring this equipment to every outdoor node deployment.

---

## Tools

| Tool | Use |
|------|-----|
| Cordless drill + bits | Mounting holes, cable gland holes |
| Step drill bit (6-22mm) | Clean holes in ABS/polycarbonate enclosures |
| Adjustable wrench (2) | U-bolts, cable glands |
| Phillips/flathead screwdriver set | Enclosure screws, terminal blocks |
| Wire strippers | Cable preparation |
| Cable ties (UV-rated, black) | Cable management |
| Electrical tape | Insulation, temporary wrapping |
| SMA torque wrench (8 in-lb) | Antenna connector tightening (do not over-torque) |
| Multimeter | Voltage checks, continuity |
| Level (small) | Ensure enclosure is mounted level |
| Tape measure | Mounting dimensions |
| Pencil/marker | Marking drill points |
| Headlamp | Working in dim spaces |

## Consumables

| Item | Use |
|------|-----|
| Marine silicone sealant + gun | Waterproofing cable glands and seams |
| Desiccant packets (x4) | Moisture control inside enclosure |
| Dielectric grease | SMA and coax connectors (prevents corrosion) |
| Rubbing alcohol + wipes | Surface prep before adhesive/sealant |
| Tapcon screws (assorted) | Concrete/brick mounting |
| Stainless steel screws (assorted) | Wood mounting |
| Nylon washers | Prevent galvanic corrosion between dissimilar metals |

## Networking

| Item | Use |
|------|-----|
| Laptop with meshtastic CLI | Configuration and verification |
| USB-C cable (2m) | Connect to device for config |
| Pre-configured spare device | Replacement if primary fails |

## Safety

| Item | Use |
|------|-----|
| Hard hat | Working at height |
| Safety glasses | Drilling |
| Work gloves | Sharp edges, mounting hardware |
| Sunscreen | Rooftop work |
| Water | Hydration |
| First aid kit | Basic medical |
| Fall protection harness | If working near roof edge (>6 ft) |

---

## Pre-Deployment Checklist

Before leaving for the site:

- [ ] Device fully charged and configured
- [ ] Profile applied and verified (meshtastic --info)
- [ ] Channel PSK set and tested (message received by another node)
- [ ] Firmware version v2.7.15+ confirmed
- [ ] Antenna tested (not a dummy load)
- [ ] Enclosure assembled with cable glands fitted
- [ ] Solar panel tested (voltage output in sunlight)
- [ ] Battery fully charged
- [ ] All cables and adapters present
- [ ] Site survey checklist completed for this location
- [ ] Permission/access confirmed for today
- [ ] Weather check (don't install in rain/snow if possible)
