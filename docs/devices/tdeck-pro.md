# T-Deck Pro

Portable mesh client with full keyboard and display.

## Variants

### T-Deck Plus (Full-Featured) - $77-82

| Property | Value |
|----------|-------|
| Display | 2.8" IPS LCD |
| MCU | ESP32-S3 |
| LoRa | SX1262 at 22 dBm |
| GPS | Yes |
| Battery | 2000 mAh |
| Keyboard | QWERTY with backlight |
| Dual-boot | M5Stack Launcher (SD card) |

Best firmware support of any T-Deck variant. Supports multi-boot for Meshtastic/MeshCore switching.

### T-Deck Pro (E-Ink) - $92-111

| Property | Value |
|----------|-------|
| Display | 3.1" e-paper touchscreen (320x240) |
| MCU | ESP32-S3 |
| LoRa | SX1262 at 22 dBm |
| GPS | u-blox |
| Battery | 1400 mAh |
| Variants | Audio ($92-101), 4G LTE ($103-111) |

Sunlight-readable, excellent battery life. Meshtastic BaseUI works as of late 2025.

## Role in LA-Mesh

Mobile client devices for community members. Full keyboard enables direct encrypted messaging in the field.
